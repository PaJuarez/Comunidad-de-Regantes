# Decisiones de diseño

## 2026-07-05

### Encabezados

Todas las hojas utilizadas por el proyecto deben tener los encabezados en la primera fila.

Motivo:
Simplifica el código de acceso a datos y mejora el rendimiento.

Mejora futura:
Permitir configurar la fila de encabezados o detectarla automáticamente.
